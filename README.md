# react-scroll-up
React Component for a fixed scroll to top button.


## Install

```npm
npm install react-scroll-up-button
```

## Usage

###Default Button:
```jsx
import React from "react";
import ScrollUp from "react-scroll-up-button";

export default class Index extends React.Component {
    render() {
        return (
            <div>
                <ScrollUpButton />
            </div>
        );
    }
}

```
